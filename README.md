# AI SDR Agent

An AI-powered Sales Development Representative (SDR) Agent built using the OpenAI Agents SDK and Groq-hosted LLMs.

## Overview

This project demonstrates how AI agents can automate the creation of professional sales outreach emails. It explores the core capabilities of the OpenAI Agents SDK, including structured outputs, tool usage, guardrails, and Model Context Protocol (MCP).

The notebooks were adapted to use Groq-hosted models instead of the default OpenAI models.

## Features

- AI-powered cold email generation
- Multi-agent workflows
- Structured outputs with Pydantic
- Function calling and tools
- Guardrails for response validation
- MCP (Model Context Protocol) integration
- Groq LLM integration

## Tech Stack

- Python
- OpenAI Agents SDK
- Groq API
- Pydantic
- Jupyter Notebook
